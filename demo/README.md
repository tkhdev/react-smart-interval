# react-smart-interval Demo

This is the demo site for `react-smart-interval` package, built with React + Vite.

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the demo.

## Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This demo is configured to deploy on Vercel. See [DEPLOY.md](./DEPLOY.md) for detailed instructions.

### Quick Deploy

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

## Structure

- `src/` - React source code
  - `App.tsx` - Main application component
  - `components/` - Example components
  - `main.tsx` - Entry point
  - `index.css` - Global styles
- `index.html` - HTML template
- `vite.config.ts` - Vite configuration
- `package.json` - Dependencies and scripts
